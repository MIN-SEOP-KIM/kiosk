# 키오스크(kiosk)
공차 kiosk 를 생각해서 만들어봄.
# 사용언어
javaFX + Scenbuilder 사용
# 프로그램 설명
 버튼을 클릭하면 메뉴가 추가 되어 계산 버튼을 누르면 결과 값이 나오는 형식
